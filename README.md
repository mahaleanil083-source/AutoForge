# AutoForge 🚀

AutoForge is a Python desktop application for file organization and duplicate file management.

## Features

- 📂 Smart File Organizer
- 🔍 Duplicate File Finder
- 🖼 Image Preview
- 📊 File Statistics
- 📄 CSV Export
- ⚡ Modern CustomTkinter UI

## Tech Stack

- Python
- CustomTkinter
- Pillow (PIL)
- Pathlib
- CSV

## Installation

```bash
pip install -r requirements.txt
python gui.py
```
