# 🚀 AutoForge

AutoForge is a Python-based automation toolkit that helps organize files and detect duplicate files automatically.

This project is built to solve common file management problems using clean, modular, and production-ready Python code.

---

# ✨ Features

- 📁 Organize files by file type
- 📄 PDF, Images, Excel, Word, Audio, Video support
- 🔍 Duplicate file detection using SHA-256 Hash
- 🏗 Modular project architecture
- 🐍 Built with Python
